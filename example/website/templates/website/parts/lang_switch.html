{% load i18n %} {% comment %} Inline language switch. Reads ``page.alternate_urls`` (returned by ``AbstractPage.alternate_urls()``) — a dict ``{lang_code: url_or_none}`` for every language EXCEPT the currently active one. ``url`` is ``None`` when the page isn't public in that language; we render that as a disabled chip so editors / readers see the lang exists but can't follow the link. {% endcomment %}
