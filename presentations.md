# Presentations

Slide decks about camomilla releases and features — useful for briefing a dev team before an upgrade. Each deck is a single self-contained HTML page: navigate with ←/→, jump with Home/End, or print it to PDF (one slide per page).

| Deck | Covers |
|---|---|
| Release update 6.4.0 → 6.5.0 | Media renditions, Meta Models, typed `Permalink` links, the page lifecycle (drafts / preview / scheduling / revisions), and the `camomilla_makemigrations` upgrade path |

::: info For contributors
Deck sources live in [`docs/public/presentations/`](https://github.com/camomillacms/camomilla-core/tree/master/docs/public/presentations) — one self-contained HTML file per deck (inline CSS/JS, no external assets), named `YYYY-MM-<topic>.html` so they sort chronologically. Keep decks source-verified against the tagged release they present, then add a row to the table above.
:::
