# Presentations

Slide decks about camomilla releases and features — useful for briefing a dev team before an upgrade. Each deck is a single self-contained HTML page: navigate with <kbd>←</kbd>/<kbd>→</kbd>, jump with <kbd>Home</kbd>/<kbd>End</kbd>, or print it to PDF (one slide per page).

::: info For contributors
Deck sources live in [`docs/public/presentations/`](https://github.com/camomillacms/camomilla-core/tree/master/docs/public/presentations) — one self-contained HTML file per deck (inline CSS/JS, no external assets), named `YYYY-MM-<topic>.html` so they sort chronologically. Keep decks source-verified against the tagged release they present, then add a row to the table above.
:::
